# 2026-07-02

## NOTHING UI v1.0 → v1.2：层次感增强 + 微圆角

### 变更动机
用户反馈希望有"更复杂的层次感"以及"稍微圆角一点"。

### 具体变更

**1. Surface 系统从 4 层扩展到 6 层（更细粒度递进）**
- Surface 0-5，色差从 ~7L* 缩小到 4-7L* 步进
- 每层增加白色叠加（0%→16%），模拟 MD3 tonal elevation
- 新增 L* 亮度列和 ΔL 差列，量化层次感

**2. 全新阴影系统（多层层叠 + 渐进模糊）**
- 5 级阴影 token（--shadow-0 ~ --shadow-5）
- 低层（0-3dp）：纯硬阴影，无 blur
- 中层（6dp）：引入 8px 微小模糊
- 高层（12-24dp）：引入 16-24px 模糊
- 通过增加阴影层数和偏移量来增强深度感

**3. 全局圆角从 0px 改为 2px**
- 新增 --radius-none/sm/md/full 体系
- 默认组件圆角 2px（按钮、卡片、搜索栏等）
- 浮层（对话框）使用 4px
- 更新了 7 个组件的 CSS、Flutter 代码、MD3 映射、设计令牌 JSON

**4. 更新范围**
- 设计哲学对比表、第 2 章（颜色系统）、第 5 章（全部组件 CSS）、第 9 章（暗色模式 6 层反转）、第 12 章（Flutter 代码 + 设计令牌 JSON + MD3 对接说明 + 最小实现清单）、第 13 章（设计示例）、附录 A（新增阴影/形状表）
- 文档从 2190 行扩展至 2288 行
